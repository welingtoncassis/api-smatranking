
# SMARTRANKING
## Description
Api de gestão de partidas e ranking de jogadores de tênis

## Features
- Ranking
- Agendamento de Partidas
- Dasafiar outro jogador
- Histórico de partidas
- Notificação de desafio por e-mail

## Techs
- NestJS [https://nestjs.com](https://nestjs.com/)
- MongoDB
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
